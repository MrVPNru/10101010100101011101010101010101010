# 🔐 CYBER-CODER v2.0 (Ультра-Кириллица)

Легковесный и быстрый Bash-скрипт для кодирования текста в двоичный формат (нули и единицы) и обратно. Идеально работает с UTF-8 и русским языком.

## ✨ Особенности

*   **Полная поддержка кириллицы**: Корректно шифрует русский текст без потери байтов.
*   **Красивый вывод**: Разделяет байты дефисами для удобного чтения.
*   **Стресс-устойчивость**: Проверено на длинных текстах (более 100 слов).
*   **Автономность**: Не ломает терминал благодаря скрытию курсора и обработке выхода.

## 🚀 Быстрый запуск

Запусти скрипт на любом Linux-дистрибутиве (Debian/Ubuntu) одной командой:

```bash
bash <(curl -sL https://githubusercontent.com)
```

## 📦 Зависимости

Для работы скрипта используются стандартные системные утилиты:

*   `xxd` — для преобразования текста в бинарный вид.
*   `perl` — для мгновенного обратного декодирования в UTF-8.

Если они не установлены, выполни:
```bash
sudo apt update && sudo apt install -y xxd perl
```

## 🛠️ Как это работает

1.  **Шифрование**: Берет строку, переводит каждый символ в двоичный код через `xxd` и заменяет пробелы на дефисы.
2.  **Расшифровка**: Удаляет дефисы, склеивает поток нулей и единиц и упаковывает обратно в текст через `perl`.
